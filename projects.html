AutoFileOrganizer

AutoFileOrganizer is an automation tool designed to efficiently manage and organize files based on their type, date, or custom categories. It automatically sorts and moves files into designated folders, reducing clutter and improving file management on your system. Built using Python, this project integrates file handling and scheduling mechanisms, making it a practical solution for enhancing productivity and keeping your workspace tidy.


AutoTrashCleaner

AutoTrashCleaner is a simple yet efficient Python script designed to automatically delete all files in the trash folder. By scheduling the script to run at regular intervals, users can automate the cleanup process, keeping their system clutter-free without manual intervention. It's a lightweight solution for maintaining a cleaner and more organized storage space.
